# ETFClarity

ETFClarity is a data pipeline and analysis tool that makes 
JSE-listed ETF investing understandable for ordinary South Africans.

Most investment platforms show you the numbers. ETFClarity explains 
what they mean — in plain language, with real South African context.

## What it does
- Fetches and stores historical ETF price and performance data from 
  the JSE
- Translates financial metrics (TER, NAV, returns) into plain English
- Models real scenarios: "If I invest R500/month for 5 years, what 
  do I end up with?"
- Updates automatically so the data stays current

## Why it exists
Millions of South Africans have access to investment platforms like 
EasyEquities but lack the financial background to confidently choose 
between funds. ETFClarity bridges that gap.

## Tech stack
- Python · pandas · PostgreSQL · yfinance · APScheduler
